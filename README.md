# 📝 To-Do App (Vanilla JavaScript)

Aplicación web de lista de tareas desarrollada con **JavaScript puro**, que permite gestionar tareas de forma dinámica con persistencia en el navegador.

---

## 🚀 Funcionalidades

* ➕ Agregar nuevas tareas
* ✔ Marcar tareas como completadas
* ❌ Eliminar tareas
* 🔍 Filtrar tareas en tiempo real
* 💾 Guardado automático en el navegador (localStorage)
* 📊 Contador de tareas completadas

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3 (Con las librerias de Bootstrap 5)
* JavaScript (Vanilla JS)

---

## 🧠 Conceptos aplicados

Este proyecto pone en práctica:

* Manipulación del DOM
* Manejo de eventos
* Arrays y objetos en JavaScript
* Renderizado dinámico
* Filtrado de datos (`filter()`)
* Persistencia con `localStorage`
* Separación entre datos y vista

## 💡 Cómo funciona

* Las tareas se almacenan en un array de objetos
* Cada tarea tiene:

  * `texto`: contenido de la tarea
  * `completada`: estado (true/false)
* La interfaz se renderiza dinámicamente en base al estado del array
* Los datos se guardan automáticamente en `localStorage`
